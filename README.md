# 봉곡중학교 3D 홈페이지

경상북도 구미시 봉곡중학교를 소개하는 비공식 3D 랜딩 페이지입니다.

## 구성

- `index.html`: 단일 페이지 구조
- `assets/styles.css`: 반응형 레이아웃과 애니메이션 스타일
- `assets/main.js`: Three.js 기반 3D 캠퍼스 장면, 카운트업, 카드 틸트, 타임라인 리빌
- `.nojekyll`: GitHub Pages 정적 배포용

## 자료 출처

- NEIS 학교기본정보 API
- 봉곡중학교 공식 홈페이지: 학교상징, 학교현황, 학교연혁

## GitHub Pages 배포

이 폴더를 GitHub 저장소 루트로 올린 뒤, 저장소 Settings > Pages에서 `Deploy from a branch`를 선택하고 `main / root`를 지정하면 배포됩니다.
